# 正则表达式常用符号查询

|  表达式符号  | 功能                 | 说明               |
| :-----: | ------------------ | ---------------- |
| /^abc$/ | 限制开头和结尾,只能是中间部分的字串 | 也可以只限制开头,或者只限制结尾 |
|    \    | 标记转义字符             |                  |
|    *    | 匹配前面表达式的值0次或多次     | 等价于{0,}          |
|    +    | 匹配前面表达式的值1次或多次     | 等价于{1,}          |
|    ?    | 匹配前面表达式的值0次或1次     | 等价于{0,1}         |
|   {n}   | 匹配确定的n次            | 匹配的字串必须是n个       |
|  {n,}   | 匹配至少n次             |                  |
|   \b    | 匹配单词边界             |                  |
|   \B    | 匹配非单词边界            |                  |
|   \d    | 匹配数字字符             | 等价于[0-9]         |
|   \D    | 匹配非数字字符            | 等价于[             |
